## Getting Started

1. Clone this repository to your local machine:

    ```
    git clone https://github.com/Jayesh-JainX/Hacktoberfest-23.git
    ```

2. Navigate to the project directory in cmd:

    ```
    cd Your_Projects
    cd simple text editor
    ```

3. Run the text editor from the command line:

    ```
    python text_editor.py
    ```
